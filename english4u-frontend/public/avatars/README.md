Place the Speaking examiner GLB model here:

- Default path: `/avatars/examiner.glb`
- Optional override: set `VITE_SPEAKING_EXAMINER_MODEL_URL` to another public URL.

Lip sync works best when the GLB has morph targets with common names such as
`viseme_aa`, `viseme_PP`, `viseme_FF`, `mouthOpen`, or `jawOpen`.
